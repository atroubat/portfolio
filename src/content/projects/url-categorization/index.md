Mise en place d'une solution complète de catégorisation d'URLs chez Equativ :

- **Architecture microservices** pour le traitement distribué
- **Kafka** (Confluent Platform) pour le streaming de données
- **Kubernetes** (Rancher) pour l'orchestration des conteneurs
- **CI/CD** via GitLab pour le déploiement continu
- **Monitoring** avec Grafana, Prometheus et Alertmanager
